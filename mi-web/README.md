# Proyecto Web con Backend y Frontend en Vercel

## Configuración

1. Renombrar `.env.example` a `.env` y completar con tus datos de PostgreSQL.
2. Subir todo el proyecto a GitHub.
3. Conectar Vercel al repositorio.
4. Configurar en Vercel las variables de entorno:
   - POSTGRESS_USER_QUERY
   - POSTGRESS_PASSWORD_QUERY
   - POSTGRESS_DB_QUERY
   - POSTGRESS_HOST
5. Deploy.

## Uso

1. Abrir la URL de tu proyecto desplegado.
2. Ingresar nombre de usuario en la caja de texto y presionar "Cargar Productos".
3. La tabla se llenará automáticamente con los productos desde tu PostgreSQL.
